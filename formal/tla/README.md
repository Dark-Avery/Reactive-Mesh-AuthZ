# Формальная проверка

Этот каталог содержит минимальную TLA+ спецификацию жизненного цикла stream:

- `New`
- `Active`
- `RevokedPending`
- `Terminated`
- `Denied`

Локальный запуск TLC:

```bash
./scripts/run_tla.sh
```

Скрипт скачивает `tla2tools.jar` в этот каталог, если файла ещё нет, и сохраняет результат TLC в:

- локальный каталог `results/`, который не входит в GitHub-версию репозитория
